<h1>기능 설명</h1>
<h2>로그인/회원가입</h2>
<p>'리액트를 다루는 기술'이라는 책에 마지막 프로젝트인 블로그 내용에서의 로그인, 회원가입을 가져다 썼습니다. </p>
<p>아직까지 제꺼의 코드라고 할 수 없어서 설명 페이지는 다 제걸로 만든 후에 바꾸도록 하겠습니다.</p>
<h2>포스트 작성하기</h2>
<p>저만의 리액트 첫 프로젝트여서 그런지 모르는 게 너무 많았습니다.</p>
<p>예를 들어 koa-body, react-form등등 지금 모르는게 너무 많아서 한번 정리가 필요할 것 같습니다.</p>
<p>현재 기능은 react-quill을 이용하여 텍스트 편집기 작성하기, 카테고리 추가하기, 가격 정하기, 제목 정하기로만 정해놨고,</p>
<p>서버에서는 텍스트 에디터에 이미지가 있을 경우, 가장 첫번째 이미지를 썸네일로 잡게 두었습니다.</p>
